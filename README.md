# Backend of PetWaky application

## NODE JS, EXPRESS, MONGODB

## Installation

Express

```bash
npm install express
```
Mongoose

```bash
npm install mongoose
```

Bcrypt

```bash
npm install bcrypt
```

JWT

```bash
npm install jsonwebtoken
```
## START APP

```bash
npm start
```

and go to  [http://localhost:3001](#http://localhost:3001)
